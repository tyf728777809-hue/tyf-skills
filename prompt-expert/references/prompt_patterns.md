# Prompt Patterns

Use these patterns as starting points. Adapt freely.

## Universal Chinese Prompt Structure

```text
生成一张[资产类型]，[主体]。
主体：[主体身份、姿态、服装/结构、表情/状态、镜头距离]。
风格：[主风格]，融合[表现技法]与[媒介/材质]。
构图：[构图方式、分镜/镜头、空间逻辑、信息密度]。
色彩：[色彩关系、主色、辅色、背景色、饱和度/明度]。
光影：[光源类型、明暗关系、阴影边缘、曝光倾向]。
材质与后期：[纹理、印刷/摄影/数字痕迹、颗粒、磨损、故障等]。
氛围：[情绪关键词、时代/文化语境]。
限制：[避免项、参考图规则、文字规则、质量要求]。
```

## Reference Image Subject Pattern

Use when the user gives an image as subject reference.

```text
参考图仅用于识别主体，不继承参考图的画风、构图、配色、光影、材质或后期效果。
生成一张[资产类型]，主体为参考图中的[人物/角色/物体]，保留其可识别特征：[发型、服饰、轮廓、姿态气质、关键配件]。
[继续写风格、构图、色彩、光影、材质、氛围、限制]
```

## Reference Image Style Reverse Pattern

Use when the user gives an image as style reference, asks why it works, or asks to reverse-engineer a reusable prompt. Output only the final prompt unless analysis is requested.
The final output may keep only one bracketed placeholder: `[在此处替换为您想要生成的主体内容]`. Replace all other bracketed template notes with concrete visual language.

```text
[在此处替换为您想要生成的主体内容]，采用[从参考图提取的主视觉风格]，融合[表现技法]与[媒介/材质纹理]。画面成分由[主体处理方式]、[背景/环境]、[图形元素/符号层]构成，采用[构图方式]与[分镜/镜头类型]，通过[空间逻辑/透视关系]建立画面张力。光影呈现[光影特质]，色调采用[色彩科学/色彩限定/冷暖关系]，整体具有[情绪与氛围]。画面带有[时代感/文化语境]，信息密度[密度描述]，留白[留白策略]，动态状态呈现[瞬时感/运动痕迹]。后期包含[数字痕迹/印刷痕迹/摄影痕迹]，并加入[符号化特征]作为抽象视觉语言。限制：不要复刻原图中的具体角色、文字、标志或情节；文字仅作为不可读的图形排版元素；保持原图的美学质感、构图逻辑、色彩关系和材质氛围。
```

## Character Poster Pattern

```text
生成一张[角色名]的人物海报。
角色以[全身/半身/近景]呈现，姿态为[动态动作]，避免普通正面站立，表情[情绪]。
画面采用[主风格]，结合[构图机制]，通过[几何切片/窗口重叠/对角线/放射线/负空间]组织画面。
色彩限定为[主色]、[辅色]和[背景色]，形成[高对比/低饱和/冷暖对冲/双色核心]。
加入[材质/后期]，例如[半调网点、Riso颗粒、RGB偏移、Pixel Sorting、纸张磨损、胶片颗粒]。
整体氛围[情绪]，具有[现代平面设计/动画分镜/电影海报/杂志封面]质感。
限制：无随机文字，避免杂乱背景，保持角色轮廓清晰，画面边缘锐利。
```

## Digital Glitch Poster Pattern

```text
生成一张[主体]的数字故障艺术海报。
画面由多个错位矩形窗口、透明视窗、几何切片和数据碎片叠加构成，主体轮廓被部分切割但仍保持清晰可辨。
核心视觉包含 Pixel Sorting、RGB 色彩偏移、横向数字噪点、扫描线、色差边缘和局部像素撕裂。
背景采用[简洁背景/天空/纯色/网格空间]，与中心区域的[高饱和色彩/强光/云层/霓虹]形成对比。
色彩以[主色]和[背景色]为核心，少量[强调色]点缀。
整体氛围[忧郁/冷峻/超现实/未来感]，具有现代平面设计与数字空间感。
限制：不要随机乱码文字，不要过度复杂导致主体消失。
```

## Minimal High-Contrast Poster Pattern

```text
生成一张极简主义高对比平面海报，[主体]。
画面采用[对角线/中轴/大留白/低角度]构图，以大面积[背景色]建立强视觉识别。
主体位于[方位]，呈现[剪影/线稿/扁平色块/赛璐璐]处理，轮廓清晰，姿态具有[孤独/神圣/速度/压迫]感。
色彩严格限定为[二色/三色]：[颜色列表]。
材质包含[纸张纹理/丝网印刷/Riso颗粒/硬边阴影/胶片噪点]。
整体氛围[情绪]，线条锋利，画面干净，无杂乱元素。
```

## Mixed Media Photography Pattern

```text
生成一件混合媒介视觉作品。
前景为[主体]的[白色线稿/半透明剪影/极简矢量/黑色墨线]，线条[纤细/粗犷/锐利]，只在[局部]保留[发光/色彩强调]。
背景为写实摄影：[场景]，[时间/天气]，[镜头/光影]，[景深/虚化/胶片]。
构图采用[荷兰角/中心/低角度/大留白/宽银幕]，前景平面线条与背景真实摄影形成强烈对比。
色彩以[主色调]为主，加入[强调色]。
整体氛围[空灵/怀旧/忧郁/梦幻/电影感]。
限制：前景和背景分层明确，不要让摄影风格污染线稿主体。
```

## Retro Print Poster Pattern

```text
生成一张现代复古平面海报，[主体]位于画面核心。
采用[Risograph/丝网印刷/胶版印刷/半调雕刻]风格，主体由[半调网点/粗黑线/套印错位/纸张颗粒]表现。
背景为[米色旧纸/纯色/粗糙纸张]，主体背后加入[几何拱门/圆形色块/对角线/放射线]。
色彩限定为[三色]，保持强对比和印刷质感。
加入少量符号元素：[条形码、编号、星形闪光、细线轨道、技术标签]。
文字如需出现，仅作为图形排版元素，避免随机乱码。
```

## Punk Collage Pattern

```text
生成一张高冲击力朋克拼贴海报，[主体]为绝对视觉焦点。
画面融合地下独立杂志、硬核朋克、粗野主义排版和波普拼贴美学。
构图采用[放射状/广角畸变/满版高密度/撕裂层叠]，周围元素向中心压迫。
色彩严格限定为[黑/白/亮橙]或[黑/白/高饱和强调色]。
光影完全扁平化，体积依靠粗犷黑色墨线、强对比色块和密集半调网点表现。
背景包含撕裂纸边、数字噪点、斑驳印刷纹理、警示符号和抽象几何色块。
限制：信息密度高但主体必须清楚，不要随机可读文字。
```

## Negative Constraint Bank

Add only the relevant constraints:

```text
避免常规正面站姿，避免摄影棚模板感，避免廉价3D质感，避免杂乱背景，避免多余主体，避免随机乱码文字，避免低清糊脸，避免过度柔焦，避免过度网红滤镜，避免颜色失控，避免参考图风格污染，保持主体轮廓清晰，保持构图稳定，保持色彩限定。
```
